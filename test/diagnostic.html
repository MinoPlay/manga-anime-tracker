🔧 Manual Test

Run tests manually from the browser console:

// Test connection (with Client ID header) const clientId = 'a86cd94144af52f931b0a3ab74455192'; await fetch('https://api.myanimelist.net/v2/anime/1', { headers: { 'X-MAL-CLIENT-ID': clientId } }) // Use CORS proxy (if running: node proxy-server.js) await fetch('http://localhost:3000/anime/1', { headers: { 'X-MAL-CLIENT-ID': clientId } }) // Configure malAPI to use proxy malAPI.configureProxy('http://localhost:3000', false); // Now all malAPI calls will try proxy first, then fallback to direct await malAPI.getAnimeRanking('all', 4) // Search anime await fetch('https://api.myanimelist.net/v2/anime?q=naruto&limit=5', { headers: { 'X-MAL-CLIENT-ID': clientId } }) .then(r => r.json()) .then(d => console.log(d)) // Get user's anime list await fetch('https://api.myanimelist.net/v2/users/TrulyYoursMino/animelist?limit=10', { headers: { 'X-MAL-CLIENT-ID': clientId } }) .then(r => r.json()) .then(d => console.log(d)) // Test malAPI wrapper - search await malAPI.searchAnime('naruto') // Test malAPI wrapper - get anime ranking await malAPI.getAnimeRanking('all', 4) // Test malAPI wrapper - get user list await malAPI.getUserAnimeList('TrulyYoursMino', null, 10) // Direct ranking endpoint await fetch('https://api.myanimelist.net/v2/anime/ranking?ranking_type=all&limit=4', { headers: { 'X-MAL-CLIENT-ID': clientId } }) .then(r => r.json()) .then(d => console.log(d)) // Show debug info malAPI.getDebugInfo()

⚠️ Troubleshooting CORS Issues

If getting "Failed to fetch" errors:

  1. Open new terminal in VS Code
  2. Run: node proxy-server.js
  3. Keep the proxy running (you'll see: "🔄 CORS Proxy Server running at http://localhost:3000")
  4. Come back to this page and click test buttons again

Why? MyAnimeList API doesn't support CORS from browsers. The proxy server adds the necessary headers.

📝 Other Common Issues

  • Empty Results: Try a different search term or check MyAnimeList API status
  • Timeout: Your internet connection might be slow. Try again in a few seconds
  • Rate Limited: MyAnimeList allows 1000+ requests/minute. Rare unless making many rapid requests
  • HTTP Server Error: Make sure you started: python -m http.server 8000
